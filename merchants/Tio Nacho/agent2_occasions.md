## Agent 3 Guidance

**For Agent 3 (Content Architect):**

1. **DO NOT create generic "food porn" content.** Tio Nacho is NOT a polished food photography brand. Food posts should feel casual, authentic, and venue-integrated (show food on the counter, with drinks nearby, or in the context of the space).

2. **Event graphics should match the existing feed style:**
   - Dark backgrounds (black or dark green)
   - Bold, centered text in brand colors (lime green, yellow-orange, red)
   - List format for event details (date, time, entertainment, address)
   - Mexican cultural graphics (flags, peppers, cacti) for relevant occasions

3. **Caption voice varies by post type:**
   - **Food posts:** Straightforward, minimal copy. Example: "It's Tuesday. You know what that means. 🌮"
   - **Event posts:** List format with vendor tags. Example: "This Friday @ Tio Nacho / Sounds by: [DJ] / Drinks by: [Bartender]"
   - **Community posts:** Warm, inclusive. Example: "Everyone is welcome to celebrate with us 💛🧡"
   - **Fun/engagement posts:** Playful, emoji-heavy. Example: "Kidding! But real tacos are here 😂🌮"

4. **Prioritize nightlife and event content over traditional restaurant marketing.** If forced to choose between a food-beauty shot and an event-energy shot, choose the event. This is what the brand's audience expects.

5. **Use recurring post formats:**
   - Taco Tuesday → Same format every week, different menu item
   - DJ Nights → Same event graphic template, different DJ/date
   - Sports parties → Same promo style as the Canelo/Crawford boxing post from the existing feed

6. **Emoji usage is HEAVY.** See Agent 1's caption examples — emojis function as bullets, emphasis, and tone markers. Use 🌮🍻🌵🥊🎧💫✨ liberally.

7. **Self-tag in every caption.** The brand consistently tags @tionacho_nyc in body copy, not just as a mention.
