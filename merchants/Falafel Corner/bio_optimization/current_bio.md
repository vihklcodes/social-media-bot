## Bio Audit

### What's Working
- **Social proof included** -- "Voted Best Falafels in Sacramento - Sac Bee" is a strong trust signal
- **Address set in IG address field** -- good for local discovery
- **Menu categories listed** -- Mediterranean, burgers, wraps
- **Catering mentioned** -- drives B2B/event business
- **Highlight covers organized** -- Desserts, Timings, Menu shows intentional structure

### What's Missing / SEO Issues
1. **No phone number** -- "Call to order" but no number provided. Defeats the purpose
2. **No website/ordering link** -- with 1,806 weekly DoorDash orders, there should be an ordering link
3. **Display name misses franchise opportunity** -- "Falafel Corner" alone doesn't indicate this is a 19+ location chain. Should specify "Downtown Sacramento" in name
4. **IG category too generic** -- "Restaurant" should be "Mediterranean Restaurant"
5. **No hours listed** -- "Timings" is in highlights but should be in bio for quick reference
6. **Emoji overuse dilutes readability** -- every line is bookended with matching emojis (🍢...🍢, 🍔...🍔) making it look templated
7. **"Delicious Wraps" is weak copy** -- every restaurant claims delicious. What kind of wraps? Shawarma? Gyro?
8. **No halal mention** -- if halal, this is a major search keyword being missed
9. **No DoorDash/delivery mention** -- despite being the highest MP weekly volume in the dataset (1,806 orders!)
10. **Following count nearly equals followers** -- 448 following vs 443 followers suggests follow-for-follow strategy
